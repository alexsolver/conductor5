Principais Problemas Identificados:
Enums não configuráveis - Prioridades, status, categorias deveriam vir do banco
Cores hard coded - Sistema de cores deveria ser configurável
Dados mock - Muitas informações simuladas em vez de vindas da API
Localizações fixas - Filiais e locais deveriam ser cadastráveis
Textos não internationalizados - Muitos textos em português fixo
Valores default - Configurações que deveriam ser por tenant
Recomendação: Implementar um sistema de configuração de metadados onde administradores possam gerenciar essas informações via interface, tornando o sistema mais flexível e adaptável a diferentes organizações.

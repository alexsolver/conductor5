Rota de Trecho

Identificação
• Ativo (SIM/NÃO)
• ID da Rota

Definição do Trecho (adição de múltiplos trechos)
• Local A – Origem (seleção de um local cadastrado)
• Local B - Destino (seleção de um local cadastrado)

Adição de Multiplos Registros 
Tabela de exibição:

•	DE (O primeiro trecho é obrigatório iniciar do Local A)
•	TRECHO (nome ou código do trecho intermediário)
•	PARA – Trecho de destino (Se for o último trecho, é obrigatório terminar no Local B)
•	AÇÃO – Botão/ação para adicionar ou remover trecho

Operação
•	Possibilidade de cadastrar múltiplos trechos sequenciais para compor a rota (grid ou lista)
•	Cada linha do grid: DE | TRECHO | PARA | AÇÃO (Editar/Excluir)

 
Área

Identificação
•	Ativo (SIM/NÃO)
•	Nome (obrigatório)
•	Descrição
•	Código de Integração (código único para integração com outros sistemas)

Classificação (deve ter o mapa integrado e mostrar no maoa as alterações em tempo real)
•	Tipo de Área (para cada tipo de área, dev peritir escolher a cor que será representada no mapa)
- Faixa CEP (permite cadastro de várias faixas de CEP e também agrupá-las)
- Shape (permite importar arquivos de shape ou exibe ferramentas diversas para desenhar o próprio shape no mapa e salvar)
- Coordenadas (permite inserir múltiplas coordenadas através de clique no mapa ou pesquisa de endereço Map API e também ordenar os pontos em polígonos (necessários, pelo menos 3 pontos de coordenadas)
- Raio (permite criar o raio a partir de uma coordenada única no mapa e definir a metragem do raio)
- Linha (permite desenhar a própria linha de fora livre no mapa)
- Importar área (permite importar área através de arquivos shape ou kml)
